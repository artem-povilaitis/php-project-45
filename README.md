## Program "Brain Games" consists of five mathematical mini-Games.
### Requirements:
 - PHP >= 8.0
 - Composer >= 2.5
 - Make >= 4.3
### Install:
1. `git clone git@github.com:artem-povilaitis/php-project-45.git`
2. `cd php-project-45`
3. `make install`
4. `Launch any of the five Games`

 - `make brain-even`
 - `make brain-calc`
 - `make brain-gcd`
 - `make brain progression`
 - `make brain-prime`
